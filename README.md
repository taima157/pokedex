# Projeto Pokédex

Pokédex criada usando ReactJS.\
Uma pokédex é uma enciclopédia dos monstros do famoso jogo Pokémon.\
Nela possuí os nomes dos mostrinhos, pontos de vida, ataque, defesa e etc.

## Como rodar o projeto?

Você pode acessá-lo no link a seguir. [Pokédex](https://taima-pokedex.netlify.app/)

## Ou rodar ele na sua máquina da seguinte maneira

Primeiramente, você precisa ter o [node](https://nodejs.org/en/) instalado na sua máquina.\
Depois clone este repositório na sua máquina.

### Dentro da pasta do projeto, rode o seguinte comando

### `npm install`

Esse comando irá instalar as dependências do projeto.

### Em seguida rode o comando

### `npm start`

Esse comando vai subir um servidor local na sua máquina\
Abra o seguinte link no seu navegador [http://localhost:3000](http://localhost:3000)
